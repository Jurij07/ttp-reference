⚙️ Systeme

Alle Kern-Systeme die das Spiel braucht. Jedes System ist ein eigenes Script-Modul in Roblox Studio.
🌀
Cultivation & Realm System
Teilweise
WasDas Herzstück. Spieler kultivieren sich durch 26 Realms, jeder mit mehreren Stages (z.B. Qi Refinement → Ultimate Origin Supreme)
WieEXP sammeln → Stage steigt → bei Stage 9: Realm Breakthrough → Heaven Tribulation überstehen → nächster Realm
WarumGibt dem Spieler ein klares Fortschrittsziel — alles andere dreht sich darum

EXP & Stage System
Teilweise
WasJede Stage braucht eine bestimmte EXP-Menge. EXP-Bedarf steigt pro Realm massiv (×8, ×60, ×400, ×2500...)
WieEXP aus NPCs, Quests, Dungeons, Daily Login. Aptitude Grade multipliziert alle EXP-Gewinne
WarumOhne Progression-Kurve gibt es keinen Anreiz weiterzuspielen
Höchste Priorität

Lifespan System
Teilweise
WasJeder Spieler hat eine maximale Lebensspanne. Bei 0 → permanenter Tod

Karma System
Teilweise
WasGut/Böse-Wert des Spielers von −1000 bis +1000
WieGute/böse Aktionen ändern Karma. Bei Heaven Tribulation: ≥+800 → halber Schaden, ≤−800 → doppelter Schaden
WarumGibt moralische Entscheidungen und beeinflusst direkt die Schwierigkeit
